# Calculator App
---
<p>Implemented a calculator app using Java/Java Swing<p>

# Functions
* Ln
* X^Y
* Sin
* Cos
* Divide
* Radical
* Procent
* Multiply
* Addition
* Factorial
* Substraction
 > Extra content
 - Ability to lock the calculator so that no buttons can be pressed
 - NightMode
